# 랩 03 — uv (요즘 가장 빠른 통합 도구)

**목표**: `uv` 하나로 가상환경 생성 + 의존성 설치 + 잠금(lock) + 실행을 끝낸다.
랩 02 에서 `venv` + `pip` 으로 여러 단계에 나눠 하던 일을, `uv` 는 명령 하나로 처리한다.

> 소요 시간 5분 · 준비물: 터미널, 인터넷 · 이 PC(Windows)에는 uv 가 이미 설치됨(`uv --version`).
> macOS 는 `brew install uv` 또는 `curl -LsSf https://astral.sh/uv/install.sh | sh` 로 설치.
> **설치 이후의 uv 명령은 두 OS 에서 완전히 동일합니다** — uv 가 OS 차이를 흡수합니다.

## 0. 개념 — uv 가 대신하는 것
`uv` 는 Rust 로 만들어져 **매우 빠르고**, 아래를 한 도구로 통합합니다.
- 파이썬 **버전 설치**(`uv python install`) — pyenv 역할
- **가상환경** 생성 — venv 역할
- **패키지 설치** — pip 역할
- 의존성 **잠금**(`uv.lock`) — 재현 보장

`pyproject.toml` 을 그대로 읽으므로 별도 설정이 필요 없습니다.

## 1. 한 방에 환경 만들기
```powershell
cd C:\repo\yscha88\rock-paper-scissors-python
uv sync
```
예상 출력:
```
Using CPython 3.13.0
Creating virtual environment at: .venv
Resolved 27 packages in 347ms
   Building rock-paper-scissors-python @ file:///.../rock-paper-scissors-python
Installed 1 package in 140ms
 + rock-paper-scissors-python==0.1.0
```
이 한 줄이 (1) `.venv` 생성 (2) `pyproject.toml` 의존성 설치 (3) `uv.lock` 생성 을
모두 해냅니다. `.python-version` 파일에 `3.13` 이 적혀 있어 uv 가 3.13 을 골랐습니다.

## 2. 실행 — 활성화 없이 바로
`uv run` 은 "그 가상환경 안에서 이 명령을 실행"합니다. `activate` 를 칠 필요가 없어요.
```powershell
uv run python -m rps 바위
uv run pytest -q
```
예상 출력:
```
당신: ROCK / 컴퓨터: SCISSORS -> WIN
........................................                                 [100%]
40 passed in 0.06s
```

## 3. 선택 의존성(웹) 포함해서 설치
```powershell
uv sync --extra web
```
`pyproject.toml` 의 `[project.optional-dependencies]` 중 `web`(fastapi·uvicorn·gunicorn)까지
설치합니다. 이제 웹 서버를 띄울 수 있습니다(→ [랩 06](06-웹서버-uvicorn-gunicorn.md)).

## 4. 잠금(lock) 파일 — 재현성
`uv sync` 가 만든 `uv.lock` 에는 **모든 하위 의존성의 정확한 버전**이 박혀 있습니다.
랩 02 에서 `pip freeze` 로 수동으로 만들던 스냅샷을, uv 는 **자동으로** 관리합니다.
누가 언제 `uv sync` 해도 이 잠금 덕분에 **똑같은 버전 조합**이 설치됩니다.

패키지를 새로 추가하면 lock 도 자동 갱신됩니다(예시):
```powershell
uv add requests            # 의존성 추가 + pyproject.toml + uv.lock 갱신
uv remove requests         # 되돌리기
```

## 5. 파이썬 버전도 uv 로
```powershell
uv python list             # uv 가 아는 파이썬들
uv python install 3.12     # 3.12 를 uv 로 설치
uv venv --python 3.12      # 3.12 로 가상환경 만들기
```

## 6. 정리
```powershell
Remove-Item -Recurse -Force .venv
# uv.lock 은 프로젝트에 두는 게 보통이지만, 실습 후 지우려면:
Remove-Item uv.lock
```

> **언제 uv 를 쓰나**: 사실상 **거의 모든 새 프로젝트**. 가장 빠르고 통합돼 있어 입문자에게도 권합니다.
